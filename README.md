# AI-Resume-Copilot
AI Resume Copilot is an AI-powered resume builder that analyzes job descriptions, generates ATS-friendly resumes, provides resume match analysis, detects missing information, offers multiple professional templates, and exports high-quality PDFs. Built with React, Flask, Python, Tailwind CSS, SQLite, and Gemini AI.
Features
📄 AI Job Description Analysis
🎯 ATS-Friendly Resume Generation
📊 Resume Match Score & Skill Gap Analysis
💬 AI Resume Assistant
📝 Multiple Resume Templates (ATS, Modern, Minimal, Fresher)
📥 High-Quality PDF Download
🔍 Missing Information Detection
📈 Resume Completeness Dashboard
🎨 Clean and Responsive User Interface
Tech Stack
Frontend: React.js, JavaScript, Tailwind CSS
Backend: Flask (Python)
AI: Google Gemini API
Database: SQLite
PDF Generation: jsPDF + html2canvas
Project Goal

The objective of this project is to simplify resume creation by combining artificial intelligence with modern web technologies. The platform provides personalized resume recommendations, improves resume quality, enhances ATS compatibility, and helps job seekers prepare resumes tailored to specific job roles.
